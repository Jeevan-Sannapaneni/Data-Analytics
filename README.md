# Data-Analytics
🔗 Business Sales Analytics – A comprehensive business analytics project analyzing 10,000+ sales transactions to uncover insights on revenue, profit, customer behavior, regional performance, and discount impact using Python and data visualization.
🔗 Smartphone Usage Productivity Analysis – Data analysis of 50,000 users exploring the relationship between smartphone usage, sleep patterns, stress levels, and work productivity; includes preprocessing, EDA, visualization, and behavioral insights using Python, Pandas, Seaborn, and Matplotlib.
