# Data-Analytics
- 🔗 [Business Sales Analytics](https://github.com/Jeevan-Sannapaneni/business-sales-analytics-dashboard) – A comprehensive business analytics project analyzing 10,000+ sales transactions to uncover insights on revenue, profit, customer behavior, regional performance and discount impact using Python and data visualization.
- 🔗 [Smartphone Usage Productivity Analysis](https://github.com/Jeevan-Sannapaneni/smartphone-usage-productivity-analysis) – Data analysis of 50,000 users exploring the relationship between smartphone usage, sleep patterns, stress levels and work productivity; includes preprocessing, EDA, visualization and behavioral insights using Python, Pandas, Seaborn and Matplotlib.
- 🔗 [Video Game Sales EDA](https://github.com/Jeevan-Sannapaneni/video-game-sales-eda) – Exploratory Data Analysis and visualization of the top 50 best-selling video games dataset from Kaggle, uncovering sales trends, platform dominance, franchise performance, publisher impact, and release year patterns using Python, Pandas, Seaborn, and Matplotlib.
- 🔗 [Restaurant Sales Data Analysis](https://github://github.com/Jeevan-Sannapaneni/restaurant-sales-data-analysis) – Exploratory Data Analysis (EDA) and business insights from the Balaji Fast Food Sales dataset, including data cleaning, preprocessing, visualization and revenue trend analysis using Python (Pandas, Matplotlib and Seaborn).
- 🔗 [Online vs Store Shopping EDA](https://github.com/Jeevan-Sannapaneni/online-vs-store-shopping-eda) – Exploratory Data Analysis of consumer shopping behavior comparing online and in-store preferences using Python and Matplotlib, including demographic insights, spending patterns, behavioral analysis and correlation visualization.
- 🔗 [Store Customer Data Analysis](https://github.com/Jeevan-Sannapaneni/store-customer-data-analysis) – Exploratory Data Analysis of retail store customer data to understand demographics, income distribution and spending behavior using Python with data visualization and statistical insights.
