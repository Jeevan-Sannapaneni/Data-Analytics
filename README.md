# Data-Analytics
- 🔗 [Business Sales Analytics](https://github.com/Jeevan-Sannapaneni/business-sales-analytics-dashboard) – A comprehensive business analytics project analyzing 10,000+ sales transactions to uncover insights on revenue, profit, customer behavior, regional performance and discount impact using Python and data visualization.
- 🔗 [Smartphone Usage Productivity Analysis](https://github.com/Jeevan-Sannapaneni/smartphone-usage-productivity-analysis) – Data analysis of 50,000 users exploring the relationship between smartphone usage, sleep patterns, stress levels and work productivity; includes preprocessing, EDA, visualization and behavioral insights using Python, Pandas, Seaborn and Matplotlib.
- 🔗 [Video Game Sales EDA](https://github.com/Jeevan-Sannapaneni/video-game-sales-eda) – Exploratory Data Analysis and visualization of the top 50 best-selling video games dataset from Kaggle, uncovering sales trends, platform dominance, franchise performance, publisher impact, and release year patterns using Python, Pandas, Seaborn, and Matplotlib.
- 🔗 [Restaurant Sales Data Analysis](https://github://github.com/Jeevan-Sannapaneni/restaurant-sales-data-analysis) – Exploratory Data Analysis (EDA) and business insights from the Balaji Fast Food Sales dataset, including data cleaning, preprocessing, visualization and revenue trend analysis using Python (Pandas, Matplotlib and Seaborn).
- 🔗 [Online vs Store Shopping EDA](https://github.com/Jeevan-Sannapaneni/online-vs-store-shopping-eda) – Exploratory Data Analysis of consumer shopping behavior comparing online and in-store preferences using Python and Matplotlib, including demographic insights, spending patterns, behavioral analysis and correlation visualization.
- 🔗 [Store Customer Data Analysis](https://github.com/Jeevan-Sannapaneni/store-customer-data-analysis) – Exploratory Data Analysis of retail store customer data to understand demographics, income distribution and spending behavior using Python with data visualization and statistical insights.
- 🔗 [Loan Risk Prediction](https://github.com/Jeevan-Sannapaneni/loan-risk-prediction) – Exploratory Data Analysis of a loan risk dataset to identify key factors influencing loan approval using Python, data visualization, and statistical insights.
- 🔗 [Market Basket Analysis](https://github.com/Jeevan-Sannapaneni/market-basket-analysis) – Exploratory Data Analysis and Market Basket Analysis on a retail transactional dataset to discover purchasing patterns and product associations using Python, Pandas, and the Apriori algorithm.
- 🔗 [Uber Rides Data Analysis](https://github.com/Jeevan-Sannapaneni/uber-rides-data-analysis) – Exploratory Data Analysis and visualization of the Uber Drives 2016 dataset to analyze ride patterns,trip purposes,travel distances and time-based demand using Python, Pandas, Matplotlib and Seaborn.
- 🔗 [Udemy Courses Analysis](https://github.com/Jeevan-Sannapaneni/udemy-courses-analysis) – Exploratory Data Analysis and visualization of the Udemy courses dataset to analyze subscribers, reviews, pricing trends, course levels, and subject distribution using Python, Pandas, Matplotlib, and Seaborn.
- 🔗 [Global Billionaires Data Analysis](https://github.com/Jeevan-Sannapaneni/global-billionaires-data-analysis) – Exploratory Data Analysis of a global billionaires dataset to analyze wealth distribution, top billionaires, industry dominance, country rankings and age patterns using Python, Pandas, Matplotlib and Seaborn.
- 🔗 [Equity Market EDA – Adani Ports](https://github.com/Jeevan-Sannapaneni/equity_market_eda) – Exploratory Data Analysis and visualization of stock market data for Adani Ports, including trend analysis, trading volume insights, correlation study and technical indicators such as moving averages using Python.
- 🔗 [Amazon Bestseller Books Analysis](https://github.com/Jeevan-Sannapaneni/amazon-bestseller-books-analysis) – Exploratory Data Analysis and visualization of Amazon bestselling books (2009–2019), uncovering insights on ratings, pricing, genres, and author trends using Python, Pandas, Matplotlib, and Seaborn.
- 🔗 [Amazon Bestseller Books Analysis](https://github.com/Jeevan-Sannapaneni/amazon-bestseller-books-analysis) – Exploratory Data Analysis and visualization of Amazon bestselling books (2009–2019), uncovering insights on ratings, pricing, genres, and author trends using Python, Pandas, Matplotlib, and Seaborn.
- 🔗 [Spotify Top 100 Analysis](https://github.com/Jeevan-Sannapaneni/spotify-top100-analysis-) – Exploratory Data Analysis and visualization of the Spotify Top 100 songs dataset, uncovering insights on genres, countries, audio features and streaming trends using statistical analysis, boxplots, bar charts, pie charts and correlation heatmaps.
- 🔗 [Social Media Impact Analysis](https://github.com/Jeevan-Sannapaneni/social_media_impact_analysis) – Exploratory Data Analysis of social media data to evaluate the impact of content, user behavior and engagement metrics such as likes, shares and comments using Python, visualization and statistical insights.
